# Fitness App v1.0

Aplicacion web para organización de entrenamientos. Incluye autenticación de usuarios, dashboard con los diferentes servicios
y módulo de rutinas con su primera versión en funcionamiento.

## Características implementadas

Autenticación completa (Login/Register con JWT)
Dashboard con 5 servicios (Rutinas, Nutrición, Objetivos, Estadísticas, Perfil)
Diseño de la aplicación
Diseño responsive dependiendo del tamaño del dispositivo
API REST con Nodejs y PostgreSQL

# Fitness App v1.1

Mejora de funcionamiento del apartado de rutinas, funciona correctamente lo siguiente:
    - Apartado rutina semanal
    - Apartado de creación de entrenamientos
	- Cada entrenamiento tiene una serie de ejercicios y estos una serie de repeticiones y peso en kg
    - Apartado de visualización de entrenamientos con posibilidad de borrado

# Fitness App v1.5

Apartados objetivos, nutricion, perfil, rutinas, estadisticas implementados en la aplicacion.
Objetivos:
    - Tabla de objetivos registrados por el usuario
    - Formulario para crear un nuevo objetivo, con fecha inicial y fecha de finalizacion

Nutricion:
    - Grafica de los ultimos siete dias de datos de la base de datos asociados al usuario
    - Apartado de comidas donde se agregan las comidas que el usuario almacena
    - Formulario en el que se indica el nombre de la comida, macros y se agrega al apartado de comidas indicando el tipo de comida
    - Apartado de calorias totales por dia, proteina, carbohidratos y grasas

Perfil:
    - Opcion de poner una foto de perfil para el usuario
    - Opcion de cambiar el nombre de usuario
    - Opcion de cambiar la contraseña verificando que sabe la anterior
